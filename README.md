# Tractr

A habit tracker web app for genZ to help you build good habits and break bad ones.

Visit the live site at [tractr.app](https://tractr.vercel.app).

## Tech Stack

- Remix.js
- React
- Tailwind CSS
- Vercel

## Local Development

Clone the repo and install dependencies:

```sh
git clone
cd tractr
npm i
```

Start the dev server.

```sh
npm run dev
```
